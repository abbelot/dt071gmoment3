# DT071G - Moment 3
## Amanda Björk
### Om repot
Repot innehåller en simpel applikation i form av en gästbok skapad med C#. Användare kan lägga till och radera inlägg i gästboken. Applikationen är konsolbaserad och inläggen lagras i en JSON-fil.

### Installation
För att köra applikationen måste .NET vara installerat på maskinen. Klona repot med följande kommando: `git clone https://github.com/abbelot/dt071gmoment3.git`. Navigera till repots mapp och starta 
applikationen med `dotnet run` om applikationen körs i VS Code eller `F5` i Visual Studio för att kompilera och köra applikationen.
